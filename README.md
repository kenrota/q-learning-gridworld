# q-learning-gridworld

Simple gridworld game using Q-learning.

# Demo

![Demo](demo.gif)

- Red: Agent
- Green: Start (Reward = 0)
- Blue: Goal (Reward = 1)
- Black: Hole (Reward = -1)
- Beige: Land (Reward = 0)

# Tested Environments

- macOS 10.13.6 (High Sierra)
- Java 1.8.0_131
- JRuby 9.2.0.0 (2.5.0)

# Run

```
$ jruby main.rb
```
