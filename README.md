# q-learning-gridworld

Simple gridworld game using Q-learning.

# Demo

![Demo](demo.gif)

- Red: Agent
- Green: Start (Reward = 0)
- Blue: Goal (Reward = 1)
- Black: Hole (Reward = -1)
- Beige: Land (Reward = 0)

# Tested Environments

- macOS Sequoia
- JRuby 10.0.2.0 (Ruby 3.4 compatible)

# Run

```
$ jruby main.rb
```
